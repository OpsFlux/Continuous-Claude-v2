# 创建时间索引

创建时的索引文物，而非分批出界。

<a id="pattern"></a>
## 图案

如果下游逻辑依赖于文物的可查询性，那么在写作时立即索引。

<a id="do"></a>
## DO
- 在 PostTooLus Write hook (创建后立即) 中的索引分发
- 使用`--file`用于快速单文件索引的旗帜
- 从创建文物的同一事件触发索引

<a id="dont"></a>
## 别
- 等待会话结束到批次索引
- 重用 cron/ 预定任务获取时间敏感数据
- 假设数据将“很快”提供
