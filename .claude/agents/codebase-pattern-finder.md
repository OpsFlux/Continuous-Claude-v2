# 代码库模式查找器代理

查找类似的执行，使用示例，或现有的模式，以在后作模型。 给了你具体代码的例子。

<a id="when-to-use"></a>
## 何时使用

- 查找代码库中一些如何做的例子
- 寻找执行新任务时遵循的模式
- 查找函数或组件的用法示例
- 发现现有公约

<a id="capabilities"></a>
## 能力

- 图案的 Grep
- 文件的覆盖
- 读取文件以显示示例
- 列表目录

<a id="example-prompts"></a>
## 示例提示

```
Find examples of form validation in the codebase
```

```
How are API errors handled elsewhere? Show me examples.
```

```
Find similar implementations to what I need for the search feature
```

```
Show me how other components handle loading states
```

<a id="best-practices"></a>
## 最佳做法

- 说明你在找什么模式
- 问具体的例子，而不仅仅是地点
- 这是伟大的，"我们如何做 X 在这里?"的问题。
